# Adidas US Interactive Sales Project

## Overview
This project analyzes Adidas sales data in the United States.  
It helps understand sales performance, trends, and profit using data analysis and dashboards.

## Project Objectives
- Analyze Adidas US sales data
- Find sales trends and patterns
- Identify top products and regions
- Support better business decisions

## Dataset
- File: `Adidas-US-Interactive-Sales-Project.xlsx`
- Contains:
  - Order date
  - Product category
  - Retailer
  - Region
  - Units sold
  - Total sales
  - Profit

## Tools Used
- Microsoft Excel
